            To list bills for reprint
            **************************
            
                API - pos/reprint
                Method :GET
                Parameters :
                    api-key
                    devices_key
                    user_id
                    group_id
                    warehouse_id
                    date [format - yyyy-mm-dd]
                    bill_no[optional]
                    type [all, 0, 1] [0-print,1-dont print]

                Method: GET
                Parameters:
                    api-key
                    devices_key
                    user_id
                    group_id
                    warehouse_id
                    bill_id
